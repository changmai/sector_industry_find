# 시뮬레이션 BAT 파일 모음

## 개요
`combination_tester.py`의 다양한 옵션 조합 예시를 제공합니다.
각 BAT 파일 내부에 전략 논리와 설정 값에 대한 설명이 포함되어 있습니다.

## 파일 목록

| 파일 | 전략 | 핵심 조건 | 특수 옵션 |
|------|------|----------|----------|
| 01_섹터모멘텀_기본 | 섹터 모멘텀 + 정배열 | A, D | 기본 TP/SL |
| 02_눌림목_세력진입 | 세력 진입 후 조정 매수 | H | 양봉 필터 |
| 03_급등주_조정매수 | 급등 후 5일선 조정 | I | 이격도 필터 |
| 04_기술적신호_MACD_RSI | MACD 골든크로스 + RSI | GT_A, GT_B, GT_C | 필수조건(--required) |
| 05_거래량급증_신고가 | 거래량/대금 급증 + 신고가 | B, E, F | 돌파 임박 |
| 06_트레일링스탑_모멘텀 | 추세 추종 + 트레일링 | A, D, G | --trailing-start/offset |
| 07_타임컷_단기스윙 | 단기 모멘텀 + 타임컷 | A, C | --time-cut-days/min-return |
| 08_TPSL_최적화탐색 | TP/SL 최적값 탐색 | A, D, H | --optimize |
| 09_파라미터_자동최적화 | 2단계 파라미터 최적화 | H, I | --auto-optimize |
| 10_복합전략_전체탐색 | 7개 조건 전체 조합 탐색 | A,B,D,E,F,G,H | 91개 조합 |

## 사용 방법

1. 원하는 BAT 파일을 더블클릭하여 실행
2. 결과는 콘솔에 출력되며, `--log` 옵션으로 `logs/` 폴더에도 저장됨

## 커스터마이징

각 BAT 파일을 텍스트 편집기로 열어 다음 값들을 수정할 수 있습니다:

```batch
--conditions A,D,H          # 테스트할 조건
--start-date 20240101       # 시작일
--end-date 20241231         # 종료일
--tp 10                     # 익절선 (%)
--sl -5                     # 손절선 (%)
--max-holding 14            # 최대 보유일
--min-trades 50             # 최소 거래건수
```

## 조건 설명

### 기본 조건 (A~I)
- **A**: 섹터+업종 N일 연속 상승
- **B**: N일 신고거래대금
- **C**: 거래량 회전율 상위 N종목
- **D**: 이평 정배열 (단기>중기>장기)
- **E**: N일 신고가 대비 등락률 범위
- **F**: N일전 대비 거래량 비율
- **G**: 회전율 X%~Y% 범위
- **H**: 20일선 눌림목 (세력진입 후 조정, 양봉필터)
- **I**: 5일선 급등주 (모멘텀 조정, 양봉필터)

### GT 조건 (기술적 지표)
- **GT_A**: MACD 골든크로스
- **GT_B**: RSI 매수 신호 (B1 OR B2)
- **GT_C**: RSI 과매수 아님 (<70)
- **GT_D**: 바닥 패턴 (저점대비 반등)
- **GT_D2**: 변동성 축소
- **GT_E**: 이평선 수렴
- **GT_F**: 거래량 급증

## 주의사항

- 백테스트 결과는 과거 데이터 기반이며, 미래 수익을 보장하지 않습니다.
- 과적합(Overfitting)에 주의하세요. Out-of-Sample 테스트를 권장합니다.
- 시뮬레이션은 슬리피지, 수수료 등이 완벽히 반영되지 않을 수 있습니다.
