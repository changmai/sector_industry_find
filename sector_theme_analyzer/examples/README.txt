combination_tester.py 실행 예시 BAT 파일 모음
================================================================================

■ 조건 목록 (conditions/default.json에서 파라미터 수정 가능)
--------------------------------------------------------------------------------
  기본 조건:
    A : 섹터+업종 N일 연속 상승
    B : N일 신고거래대금
    C : 거래량 회전율 상위 N종목
    D : 이평 정배열 (단기>중기>장기)
    E : N일 신고가 대비 등락률 범위
    F : N일전 대비 거래량 비율
    G : 회전율 X%~Y% 범위
    H : 20일선 눌림목 (세력진입 후 조정)
    I : 5일선 급등주 (모멘텀 조정)

  GT (기술적 지표) 조건:
    GT_P1  : 특정 테마(섹터) 소속 종목
    GT_M1  : 지수 MACD 상승 추세
    GT_M2  : 지수 RSI 과매수 아님
    GT_A   : MACD 골든크로스
    GT_B   : RSI 매수 신호 (B1 OR B2)
    GT_B1  : RSI 과매도 탈출 (30 상향돌파)
    GT_B2  : RSI 2일 연속 상승
    GT_C   : RSI 과매수 아님 (<70)
    GT_D   : 바닥 패턴 (저점대비 반등)
    GT_D2  : 변동성 축소
    GT_E   : 이평선 수렴
    GT_F   : 거래량 급증

■ 예시 파일 목록
--------------------------------------------------------------------------------
  01_basic_test.bat       : 기본 테스트 (고정 TP/SL)
  02_with_required.bat    : 필수 조건 지정 테스트
  03_optimize_tpsl.bat    : TP/SL 최적화 모드
  04_trailing_stop.bat    : 트레일링 스탑 사용
  05_time_cut.bat         : 타임컷 사용
  06_gt_conditions.bat    : GT (기술적 지표) 조건 사용
  07_all_options.bat      : 모든 옵션 사용 (참고용)
  08_quick_test.bat       : 빠른 단일 조합 테스트
  09_with_logging.bat     : 로그 파일 저장 (진행상황 + 결과)
  10_overnight_test.bat   : 야간 장시간 테스트용
  11_start_independent.bat: VS Code와 독립된 CMD 창에서 실행

■ 사용 방법
--------------------------------------------------------------------------------
  1. BAT 파일을 더블클릭하여 실행
  2. 또는 명령 프롬프트에서 직접 실행
  3. 결과는 콘솔에 출력됨

■ 로그 파일 저장 (--log 옵션)
--------------------------------------------------------------------------------
  --log 옵션을 사용하면:
  1. 진행 상황이 실시간으로 화면 + 로그 파일에 동시 저장
  2. 최종 결과가 JSON 파일로 저장 (분석용)

  저장 위치: sector_theme_analyzer/logs/
    - combo_test_YYYYMMDD_HHMMSS.log          (전체 진행 로그)
    - combo_test_YYYYMMDD_HHMMSS_results.json (결과 데이터)

  장시간 테스트 시 반드시 --log 옵션을 사용하세요!
  VS Code가 꺼져도 로그 파일은 남아있습니다.

■ VS Code와 독립 실행 (야간 테스트용)
--------------------------------------------------------------------------------
  11_start_independent.bat을 사용하면 새 CMD 창에서 테스트가 실행됩니다.
  VS Code를 완전히 종료해도 테스트가 계속 진행됩니다.

  또는 직접 실행:
    start cmd /k "cd /d 경로 && python combination_tester.py ... --log"

■ 주의 사항
--------------------------------------------------------------------------------
  - --optimize 모드는 시간이 오래 걸립니다
  - 조건이 많을수록 조합 수가 기하급수적으로 증가합니다
  - min-trades 이하 거래수는 무효 처리됩니다
  - 장시간 테스트 시 반드시 --log 옵션 사용 권장
